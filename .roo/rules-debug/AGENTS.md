# Project Debug Rules (Non-Obvious Only)
- Market trading detection uses Tencent index timestamp and a 9:20 weekday fallback; if debugging “no refresh” issues, verify the Tencent API response date first ([`services/api.ts`](services/api.ts:397)).
- EastMoney NAV/history and index history use script injection; failures can come from shared global state `window.apidata` or callback cleanup, so check queue serialization and cleanup when diagnosing missing data ([`services/api.ts`](services/api.ts:124)).
